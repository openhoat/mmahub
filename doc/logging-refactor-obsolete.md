# Documentation obsolète - Refactorisation du système de logging

## ⚠️ Cette documentation est OBSOLÈTE

Ce document décrit une ancienne version du système de logging qui a été remplacée par une implémentation utilisant le module Nuxt `nuxt-pino-log`.

Pour consulter la documentation à jour sur le système de logging actuel, veuillez vous référer à :
- `doc/nuxt-logging-refactor.md` - Documentation de la refactorisation actuelle avec Nuxt Pino Log

## Historique

Cette documentation décrivait les changements apportés au système de logging de l'application MMA Hub pour remplacer tous les `console.log` par le logger centralisé et mettre en place un système d'injection de dépendances cohérent.

Cette implémentation a été remplacée le 02/11/2025 par une version utilisant le module Nuxt standard `nuxt-pino-log` pour une meilleure intégration avec l'écosystème Nuxt et une configuration plus simple.

## Pourquoi cette documentation est conservée

Cette documentation est conservée à titre historique pour comprendre l'évolution du projet, mais elle ne reflète plus l'implémentation actuelle du système de logging.
